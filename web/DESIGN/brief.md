# Project Brief: LynxMusic Web

## What We're Building
Premium web version of LynxMusic native android app. Full-featured music streaming client. High-end aesthetic, buttery smooth animations, responsive.

## Primary Target Audience
- **Music Enthusiasts**: Want clean, fast desktop/mobile web player.
- **Android Users**: Want same experience on their computers.

## Goals
1. **Trust**: Professional, polished look. No "amateur" vibes.
2. **Engagement**: Intuitive navigation, immersive player.
3. **Retention**: High-quality audio, saved playlists, synced lyrics.

## Requirements
- Fully responsive (mobile-first approach).
- Dark mode primary (Spotify-esque but unique).
- Performance optimized (Vite/React).
- Accessible (Keyboard navigation, Screen readers).

## Sections
1. **Sidebar Navigation**: Fixed left (desktop), Bottom nav (mobile).
2. **Main Content**: Dynamic routes for Home, Search, Playlists.
3. **Global Player**: Immersive bar with progress, volume, queue.
4. **Fullscreen Player**: Cinematic view with synced lyrics.
