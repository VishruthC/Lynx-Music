# Style Guide: LynxMusic Web

## Design Philosophy
"Cinematic Minimalism". Dark, high-contrast, deep shadows, glassmorphism accents. High trust, sleek.

## Color Palette

### Primary
- **Main**: #FFFFFF (High visibility text/icons)
- **Secondary**: #B3B3B3 (Metadata/Subtitles)
- **Accent**: #00E5FF (Vibrant Cyan - for active states and highlights)

### Neutral
- **Background**: #000000 (Pure black)
- **Surface**: #121212 (Elevated components)
- **Card**: #181818 (Secondary elevation)
- **Border**: #282828 (Subtle separators)

## Typography

### Font Families
- **Headlines/Body**: 'Inter', system-ui, sans-serif.

### Scale
- **Display**: 48px / 1.1 (Hero titles)
- **H1**: 32px / 1.2
- **H2**: 24px / 1.3
- **Body**: 14px / 1.5
- **Small**: 12px / 1.5

## Spacing System
- 8px base grid.
- **XS**: 4px
- **SM**: 8px
- **MD**: 16px
- **LG**: 24px
- **XL**: 32px

## UI Elements
- **Border Radius**: 8px standard, 50% for circles.
- **Glassmorphism**: 10px blur on player bars/overlays.
- **Shadows**: `0 8px 24px rgba(0,0,0,0.5)` for cards.

## Animation
- **Hover**: 0.2s ease (opacity/scale).
- **Transitions**: 0.3s ease-in-out (page fades).
- **Lyrics**: Smooth scroll sync.
